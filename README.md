# How to add a reusable rich text editor in your React app using Draftjs and react-draft-Wysiwyg.
## Overview.
- This applications shows how to add a rich text editor from draft-js and react-draft-wysiwyg in a React App.

## Requirements
- [Nodejs](https://nodejs.org/en/download/) installed on your machine.
- [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git) installed on your machine.
- [Yarn](https://classic.yarnpkg.com/en/docs/install/#mac-stable) or [npm](https://www.sitepoint.com/beginners-guide-node-package-manager/) installed.
- Some knowledge about [React](https://reactjs.org/docs/getting-started.html) Apps(if you are a beginner still donâ€™t worry, try to catch up).

## Get started
- Clone this repo on your machine by typing `git clone https://github.com/learningdollars/haddy-react-draftjs-wysiwyg`. 
- While in the root directory of the folder; 
- Run
`yarn` This installs all the dependecies so as to get us started. Then run
`yarn start` to start the app

### `yarn start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.
